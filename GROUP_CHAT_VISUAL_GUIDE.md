## Mobile Responsive Behavior

### Desktop (> 768px)
- 3-column member grid
- 4-column media grid
- Side-by-side stats

### Mobile (< 768px)
- 1-column member grid
- 2-column media grid
- Stacked stats
- Scrollable tabs

## Keyboard Shortcuts (Future Enhancement Ideas)

- `Ctrl/Cmd + I` - Open Group Info
- `Ctrl/Cmd + /` - Focus message input
- `Tab` - Switch between tabs in Group Details
- `Esc` - Close modals/go back
